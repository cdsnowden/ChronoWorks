# ChronoWorks - Screenshot Guide for Google Play Store

## Requirements

**Google Play Store Screenshot Requirements:**
- **Minimum:** 2 screenshots (required)
- **Maximum:** 8 screenshots (recommended)
- **Dimensions:** Minimum 320px on shortest side, maximum 3840px on longest side
- **Recommended Size:** 1080 x 1920 pixels (9:16 ratio) or 1080 x 2340 pixels
- **Format:** PNG or JPG (PNG preferred for quality)
- **File Size:** Maximum 8MB per image

## How to Take Screenshots on Android

### Method 1: Built-in Screenshot
1. Navigate to the screen you want to capture
2. Press **Power + Volume Down** simultaneously
3. Screenshot saves to Photos/Screenshots folder

### Method 2: Using Developer Tools
1. Enable Developer Options
2. Connect phone to computer
3. Use Android Studio â†’ Device File Explorer
4. Or use `adb` command: `adb shell screencap -p /sdcard/screenshot.png`

### Method 3: Using Third-Party Apps
- **Screenshot Easy** - Adds floating button for screenshots
- **Screen Master** - Screenshot + editing tools

## Screenshot Editing Tips

### Required Edits
1. **Crop to exact dimensions:** 1080 x 1920 pixels (or your chosen ratio)
2. **Remove sensitive data:** Blur/remove any real employee names, phone numbers, emails
3. **Use sample data:** Create demo employees like "John Smith", "Jane Doe", "Mike Johnson"

### Optional Enhancements
1. **Add text overlays:** Highlight key features with arrows or text
2. **Frame in device mockup:** Show screenshot inside a phone frame
3. **Add captions:** Brief text describing the feature
4. **Use consistent styling:** Same fonts, colors, layout across all screenshots

### Tools for Editing
- **Canva** (free, web-based) - Great for adding text and frames
- **Figma** (free for personal use) - Professional design tool
- **Photoshop/GIMP** - Full-featured image editing
- **Snapseed** (mobile) - Quick edits on phone
- **MockUPhone** - Add device frames around screenshots

## After Taking Screenshots

1. **Transfer to computer** if captured on phone
2. **Organize in a folder:** `ChronoWorks/Screenshots/`
3. **Name files descriptively:**
   - `01_clock_in_screen.png`
   - `02_dashboard_working_now.png`
   - `03_weekly_schedule.png`
   - etc.

4. **Upload to Google Play Console:**
   - Go to Store presence â†’ Main store listing
   - Scroll to "Screenshots" section
   - Upload in the order you want them to appear
   - Add captions for each

## Pro Tips

1. **Consistency is key** - Use the same device, font size, and styling
2. **Show real features** - Don't mock up features that don't exist
3. **Update regularly** - Replace screenshots when you add new features
4. **A/B test** - Google Play Console allows A/B testing of store listings
5. **Look at competitors** - See how similar apps present their screenshots
6. **First impression matters** - First 2-3 screenshots are most important
